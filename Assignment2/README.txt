Admin.java - The admin account file, for users who wish to become sellers of products or manage other admin.
Client.java - The main client used for the project
Customer.java - The customer account file, for users who wish to shop with the client.
Item.java - The item file, detailing everything that entails products sold on the client.
Registration.java - The registration file, dealing with registering both user and admin accounts.
ShoppingCart.java - The shopping cart file, meant for handing items customers would like to purchase.

Account.java - The abstract class to Admin.java and Customer.java

Hello.java - A test file for testing the server.
Server.java - The server for the client to run on.
ClientImpl.java - The actions that need to be run by the server.
ServerClient.java - The interface for ClientImpl.java.
